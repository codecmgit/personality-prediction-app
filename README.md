# 🧠 MBTI Personality Prediction Web App

A Machine Learning based web application that predicts a user's MBTI personality type from text input using NLP techniques and a trained classification model.

🔗 **Live App:**  
https://chirashree-codecmgit-personality-prediction-app.streamlit.app/

## 🚀 About The Project

This project uses Natural Language Processing (NLP) and Machine Learning to analyze user text and predict their MBTI personality type.
The model was trained using TF-IDF vectorization and a classification algorithm, then deployed as a live web application using Streamlit Cloud.

## 🛠 Tech Stack

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Streamlit
- Joblib
- Git & GitHub

## ⚙️ How It Works

1. User enters text in the input box.
2. Text is transformed using a trained TF-IDF vectorizer.
3. The saved ML model predicts the MBTI personality type.
4. The result is displayed with personality description and career suggestions.
5. User feedback is collected and visualized.
